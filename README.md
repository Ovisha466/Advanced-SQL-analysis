## 🚀 Project Overview

As part of an advanced SQL analytics project, I worked as a data analyst intern scenario for Major League Baseball (MLB), leveraging decades of historical player data to uncover meaningful insights.

The dataset includes detailed player-level information such as:

• Schools attended  
• Salary history  
• Team associations  
• Physical attributes (height, weight)  
• Career timelines  

This project focuses on using **advanced SQL techniques** to analyze trends, compare performance, and generate business-relevant insights.

---

## 🎯 Business Questions Answered

### 🎓 What schools do MLB players attend?

• Identified top schools producing MLB players  
• Analyzed distribution of players across institutions  
• Used aggregation and grouping logic for insights  

---

### 💰 How much do teams spend on player salaries?

• Analyzed salary trends across teams and years  
• Compared spending patterns between franchises  
• Used window functions for ranking and comparisons  

---

### 📈 What does each player's career look like?

• Built career timelines using multi-table joins  
• Tracked progression across teams and seasons  
• Used CTEs to structure step-by-step analysis  

---

### ⚖️ How do player attributes compare?

• Compared height, weight, and performance metrics  
• Identified patterns across different player groups  
• Applied filtering and segmentation techniques
